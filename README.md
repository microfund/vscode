# VSCode Tips & Useful Plugins

## 1. マウスで拡大縮小する設定

1. **Ctrl + ,** で設定を開く  
2. 検索欄に **mouse wheel zoom** と入力  
3. **Editor: Mouse Wheel Zoom** にチェックを入れる  

---

## 2. 使えるプラグイン（Mermaid系）

VSCodeで Mermaid を扱う際に便利な拡張機能：

- **Markdown Preview Mermaid Support**  
- **Mermaid Chart**  
- **Mermaid Preview**

---

## 4. その他メモ

- **ビックリマーク（!）**  
  → YAMLファイルの型表現として使われることがある（例：`!Ref` など）

- **yamlファイル形式を示している**  
  → `.yml` や `.yaml` 拡張子のファイル

- **フォルダを閉じる**  
  → エクスプローラー左側の「>」マークをクリック

---

## 5. 日本語化

1. 拡張機能から **Japanese Language Pack for Visual Studio Code** をインストール  
2. VSCode を **再起動** すると日本語化される

---

## 6. 拡張機能の設定方法

1. **ファイル**  
2. **ユーザー設定**  
3. **設定**  
4. **拡張機能**  
5. 各プラグインごとの設定を変更可能
